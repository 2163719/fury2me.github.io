## **用到的函数**   
* 1.unifrnd()：生成连续均匀的随机数，如R = unifrnd(-1,1,2,1)，生成一个2行1列的矩阵R，矩阵元素在-1到1之间随机生成   
* 2.horzcat()：水平拼接数组（矩阵），要求数组行数相等。如Fib = horzcat([0;0;0;0],[1;1;1;1]),将4行1列的矩阵水平拼接成4行2列的矩阵   
* 3.int8()：可将数据转化为8位（1字节）带符号的整数
